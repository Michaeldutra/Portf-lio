Habilidades

HTML, CSS e JavaScript

HTML, CSS e JavaScript são os três pilares fundamentais no desenvolvimento de aplicações web modernas. Essas tecnologias trabalham em conjunto para criar uma experiência interativa e visualmente atraente para os usuários.

React JS

React é uma poderosa biblioteca JavaScript utilizada para desenvolver interfaces de forma declarativa, eficiente e flexível. Com o React, é possível criar componentes reutilizáveis e combiná-los para construir interfaces de usuário complexas.

Node JS

Node.js é uma plataforma de desenvolvimento de software de código aberto e multiplataforma que permite a execução de código JavaScript fora de um navegador web. Baseado no interpretador V8 do Google, o Node.js oferece aos desenvolvedores uma forma eficiente e escalável de criar aplicativos de servidor e de linha de comando.

MySQL

O MySQL é um sistema de gerenciamento de banco de dados, que utiliza a linguagem SQL como interface. É atualmente um dos sistemas de gerenciamento de bancos de dados mais populares da Oracle Corporation, com mais de 10 milhões de instalações pelo mundo.

Angular

Angular é um framework de desenvolvimento de aplicativos web criado pela Google. Ele permite criar aplicativos de página única (SPA) complexos e interativos. Com uma arquitetura baseada em componentes, o Angular facilita a construção e a manutenção de código reutilizável.
